Abstract

Search visibility is one of the most important indicators of website performance. Organizations rely on Google Search Console and Google Analytics to monitor impressions, clicks, sessions, and engagement. However, identifying which signals contribute the most to search ranking is difficult because of the large number of interacting features.

This project develops a machine learning framework using the FlyRank warehouse dataset to analyze ranking signals and classify webpages into high-visibility and low-visibility categories. A Random Forest classifier is trained on Search Console and Google Analytics metrics after appropriate preprocessing and feature engineering.

Feature importance analysis reveals which engagement and search metrics have the strongest influence on ranking performance. The proposed pipeline provides interpretable recommendations that can help SEO teams prioritize optimization efforts.

Introduction

Google Search ranking depends on hundreds of signals including content quality, user engagement, clicks, impressions, and behavioral metrics. Understanding these relationships enables organizations to improve visibility and organic traffic.

The primary objective of this research is to investigate which measurable website signals contribute most strongly to search visibility and whether machine learning can accurately distinguish high-performing webpages from low-performing webpages.

Methodology

1. Data Collection

The FlyRank warehouse dataset integrates Google Search Console and Google Analytics 4 metrics including impressions, clicks, page views, sessions, engagement, AI referrals and user behavior.

2. Data Preprocessing

The dataset was cleaned by checking missing values, duplicate records, data types and feature consistency. Binary target labels were created based on search visibility while preventing data leakage by removing ranking variables used for target generation.

3. Feature Engineering

Important behavioral metrics, traffic sources, engagement variables and search performance indicators were selected for model training.

4. Model Training

A Random Forest Classifier was trained using an 80–20 train-test split. Model performance was evaluated using Accuracy, Precision, Recall, F1 Score and ROC-AUC.

Conclusion

This study demonstrates that machine learning can successfully identify important ranking signals using Google Search Console and Google Analytics data.

The Random Forest model provides an interpretable framework for predicting search visibility while highlighting the most influential ranking factors.

The developed workflow can assist SEO professionals in prioritizing optimization activities based on evidence rather than intuition.
